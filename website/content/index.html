Send and receive SMS text messages in your Python scripts using only a Gmail account.

Installation is easy

Simply run pip install shawk to get the latest release.

Log in with any Google account

You'll send messages to users via your email account, and they can reply inline.

Periodic message polling system

Custom periodic message retrieval system is easy to understand but powerful in practice.

Designed for the Internet of Things

Shawk has been built from the ground-up for compatibility with Raspberry Pi for the Internet of Things.

  • Raspberry Pi and I.o.T. compatible
  • Fully open sourced on GitHub
  • Keep your messages in Google's cloud
  • Actively maintained
  • Well-documented

Comprehensive documentation available online

Shawk has comprehensive documentation available online on Read the Docs,
which is fully open-sourced and automatically kept up-to-date.
